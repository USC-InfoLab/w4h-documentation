Use W4h dashboard to track your patients’ health data

Introduction

The W4H GeoMTS dashboard is designed to provide visualization and analysis capabilities for GeoMTS data catered specifically for Apple Watch demo purposes.

How to play with it

  1. Log in
    We have provided a default account for you to test the system. In the future, w4h will be polished to support multiple users and password management. The default account is: > username: admin
    password: admin

  2. Create/Manage your database instance in your DB server:
    At this moment, you should have read the Setting up tutorial and set up your database server. If you already have data in proper format in your database, you can skip this step.
    Click the “import page” button on the left side of the dashboard, you will see a page like this:
    import_page_create

    Let’s start from creating a new database instance:
    First, Select “Create new W4H database instance”
    create_new_db

    Second, type in the name of database you want to make, and click “create”. In this case we name it “test2” import_page_create If it’s created successfully, you will see a message like this:
    import_page_create

    Third, select “Choose existing W4H database instance”
    import_page_create Select the database you just created
    import_page_create

    For the Following step, if you need some test file, try to download here:
    synthetic_subject_data.csv

    synthetic_timeseries_data_reduced.csv

    Fourth, Upload your subjects csv file, and check “Populate subject table name”. import_page_create After making sure corresponding Column are all correct, click “Populate database” at the bottom.

    Fifth, upload your time series csv file
    import_page_create After making sure corresponding Column are all correct, click “Populate database” at the bottom.
    import_page_create
